指标与算法原理

深入理解 FUND-OS 背后的数学模型 · 从技术指标到估值锚定与策略回测

目录

① 趋势类指标 ② 摆动类指标 ③ 波动与量能 ④ 风险指标 ⑤ 估值锚定 ⑥ 策略回测 ⑦ 预测模型

① 趋势类指标

MA 移动平均 (Moving Average)

MA(n) = (P₁ + P₂ + … + Pₙ) / n

对最近 n 个交易日的收盘价取算术平均，平滑短期噪声以呈现趋势方向。常用 5/10/20/60/120/250 日。价格上穿均线为金叉看多，下穿为死叉看空。

EMA 指数移动平均 (Exponential MA)

EMAₜ = α·Pₜ + (1−α)·EMAₜ₋₁， α = 2/(n+1)

对近期价格赋予更大权重，比 MA 更灵敏地反映趋势变化，常用参数 12/26。

MACD 指数平滑异同移动平均

DIF = EMA(12) − EMA(26)
DEA = EMA(DIF, 9)
MACD柱 = 2·(DIF − DEA)

DIF 与 DEA 金叉为买入信号，死叉为卖出信号；红绿柱反映多空动能强弱。是趋势与动量结合的经典指标。

ADX 平均趋向指数

ADX = MA(100·|DI⁺−DI⁻| / (DI⁺+DI⁻), 14)

衡量趋势"强度"而非方向，ADX > 25 表示趋势明显，< 20 表示震荡市。

② 摆动类指标

RSI 相对强弱指标

RSI = 100 − 100 / (1 + RS)， RS = 平均涨幅 / 平均跌幅

取值 0~100，> 70 视为超买，< 30 视为超卖。用于识别价格反转机会。

KDJ 随机指标

RSV = (C−Lₙ) / (Hₙ−Lₙ) × 100
K = 2/3·Kₜ₋₁ + 1/3·RSV， D = 2/3·Dₜ₋₁ + 1/3·K， J = 3K − 2D

K、D 金叉且 J < 0 为买入信号；K、D 死叉且 J > 100 为卖出信号。

布林带 Bollinger Bands

中轨 = MA(20)
上轨 = MA(20) + 2σ， 下轨 = MA(20) − 2σ

基于均值与标准差刻画价格波动区间。价格触上轨可能超买，触下轨可能超卖；带宽收窄预示变盘。

CCI / Williams %R / ROC

CCI：价格与其统计均值偏离度，>100 超买 <−100 超卖。
Williams %R：近期最高最低价的相对位置，0~−20 超买、−80~−100 超卖。
ROC：变动率 = (Pₜ − Pₜ₋ₙ)/Pₜ₋ₙ ×100，衡量动量速度。

③ 波动与量能指标

ATR 真实波幅

TR = max(H−L, |H−Cₜ₋₁|, |L−Cₜ₋₁|)
ATR = MA(TR, n)

衡量单根 K 线真实波动幅度，常用于止损位设定（如 2×ATR 止损）与仓位管理。

OBV 能量潮 / MFI 资金流量

OBV：上涨日加量、下跌日减量，累积量能判断资金流向。
MFI：RSI 的量能版本，结合典型价与成交额，>80 超买 <20 超卖。

VWAP 成交量加权均价

VWAP = Σ(Priceᵢ × Volumeᵢ) / ΣVolumeᵢ

机构常用基准价，价格高于 VWAP 偏强、低于偏弱，适合日内交易参考。

④ 风险与组合指标

夏普比率 Sharpe

Sharpe = (Rₚ − R_f) / σₚ

超额收益与总波动之比，衡量"每承担一单位风险获得的回报"，越高越好。

索提诺 / 卡玛比率

Sortino = (Rₚ − R_f) / σ_下行
Calmar = 年化收益 / |最大回撤|

索提诺只考虑下行波动；卡玛衡量收益对回撤的覆盖能力，更关注"最坏情况"。

最大回撤 Max Drawdown

MDD = min((Vₜ − max(V₀..ₜ)) / max(V₀..ₜ)) × 100%

资金曲线从历史高点到后续最低点的最大跌幅，反映极端亏损风险。

α / β (CAPM)

Rᵢ − R_f = α + β(Rₘ − R_f) + ε

β 衡量相对市场的系统性风险敏感度；α 为跑赢市场的超额收益（选股能力）。

⑤ 估值锚定原理

历史分位估值法

分位 = (价格 < 当前价 的历史样本数) / 总样本数 × 100%

将当前价格置于历史价格分布中定位：分位 ≤ 20% 视为低估，≥ 80% 视为高估，中间为合理。是当前"相对便宜/贵"的核心判据。

支撑 / 压力锚点

取历史分布的 10% / 25% / 50% / 75% / 90% 分位数作为强支撑、弱支撑、中值、弱压力、强压力位，结合当前价给出"偏离度"提示安全边际。

动量 & 波动率

20/60 日动量反映短期与中期动能；年化波动率 = 日收益标准差 × √252，刻画价格波动剧烈程度，辅助判断估值区间的可靠性。

⑥ 策略回测原理

双均线交叉 (MA Cross)

快线(MA5)上穿慢线(MA20)为金叉，满仓买入；下穿为死叉，空仓持币。捕捉趋势中段收益，震荡市易频繁止损。

定投 DCA

每个交易日投入固定金额，价格低时自动多买份额、高时少买，长期摊平成本、降低择时风险，能显著平滑回撤。

买入持有 Buy & Hold

期初一次性买入持有至期末，作为所有主动策略的业绩基准（Benchmark）。

回测评价指标

资金曲线 = 各策略持仓市值序列；总收益、年化、最大回撤、夏普均基于该曲线计算，保证横向可比。

⑦ 预测模型原理

线性回归 LR

对净值时间序列拟合 y = a·x + b，用最小二乘法求解斜率，外推下一交易日净值。

蒙特卡洛模拟 (GBM)

Sₜ₊₁ = Sₜ·(1 + μ·dt + σ·√dt·Z)

基于历史收益均值 μ 与标准差 σ，用几何布朗运动模拟上千条未来路径，输出预期净值与 5%/95% 置信区间、上涨概率。

多模型集成

对 MA / EMA / LR 三种预测按权重融合（默认 0.3/0.3/0.4），降低单一模型偏差，提升稳健性。
